문자함수 연습

    0         1         2         3
    0123456789012345678901234567890
    ABCDEFGHIJKLMNOPQRSTUVWXYZabcde
    ABC   BCDEABC F  GHI JKLMN....ABC .....    ABC... QPO

length : 객체 또는 문자 또는 열의 길이 구하는 함수

substring(a,b) : 문자열 발췌(a에서부터 b앞까지)-시작값이 끝값보다 크면 자동으로 바꿔서 출력

slice(a,b) : 문자열 발췌(a에서부터 b앞까지)-시작값이 끝값보다 크면 출력하지 않음

substr() - substring, slice와 같음

replace(a, b) : a를 b로 치환

replaceAll(a, b) : a를 b로 모두 치환

toUpperCase() : 대문자로 모두 변환

toLowerCase() : 소문자로 모두 변환

concat(a, b) : a와 b문자열 결합

trim() : 문자열의 앞뒤(양쪽) 공백을 무시

trimStart() : 문자열의 앞쪽 공백을 무시

trimEnd() : 문자열의 뒤쪽 공백을 무시

replaceAll(' ','') : 문자열 안의 모든 공백을 무시

charAt : 하나의 문자만 발췌

charCodeAt : 문자의 아스키코드를 출력

split : 특정문자를 기준으로 잘라낸다. 이 때 잘라낸 값들은 배열변수에 기억된다.

indexOf() : 문자(열) 검색시에 사용 : 있으면 위치값을, 없으면 -1을 반환

lastIndexOf() : 문자(열) 검색시에 사용(뒤에서부터 검색) : 있으면 위치값을, 없으면 -1을 반환

search() : 문자(열) 검색시에 사용 : 있으면 위치값을, 없으면 -1을 반환

match() : 문자열 일치여부..있으면 '문자열 그대로' 반환, 없으면 '공백' 반환

includes() : 문자열 포함여부...있으면 'true' 반환, 없으면 'false' 반환
